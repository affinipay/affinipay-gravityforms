Requires:

  • Wordpress 4.1+
  • Gravity Forms 1.8.9+
  • PHP Multibyte String Extension
  • PHP Mcrypt Extension
  • SSL on the page with the credit card field.

Install/Setup:

  1. Unzip archive and copy the gravity-forms-chargeio directory into the Wordpress plugins directory.
  2. Activate the plugin.
  3. Enter in the ChargeIO API credentials. (Account Type will be selected in the ChargeIO feed later.)
  4. Select Live or Test, and save the settings.
  5. Create a Gravity form to collect payment.
  6. Create a ChargeIO feed for the form.
    • Go to Forms->ChargeIO and click "Add New"
    • This is where you tell Gravity Forms how to map the form fields to the Chargio fields. Mapping is only needed for fields other than the CC info that are required by ChargeIO to complete a transaction.
    • Select a ChargeIO account that the payment will post to.
    • Select a currency. Default is USD if no selection is made.
